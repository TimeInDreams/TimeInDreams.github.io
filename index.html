图像缩放

crop（裁剪），scale（比例放缩）

seam-carving

seam-carving缩放技术是基于裁剪最小能量的八邻域接缝

Basic Idea: remove unimportant pixels from the image

  • Unimportant = pixels with less “energy”

Seam

  • A connected path of pixels from top to bottom (or left to right). Exactly one in each row

算法流程

  1. 对每个像素计算对应能量E(i,j)，生成能量分布图Mat E
  2. 动态规划对每个像素计算累计最小能量，生成累计能量最小分布图M，M(i，j)表示从0行某个像素，到（i，j）位置像素的最小累计能量

M(i, j)= E(i, j) +min(M(i -1, j -1),M(i -1, j),M(i -1, j +1))

  1. 同时记录backtrack回溯路径图B,B(i,j)记录的值即为上一行所选择的最小能量的位置j-1,j,j+1中的一个

  2. 对整张图最后一行选择M(i,j)最小的一个位置，利用B进行像素坐标索引回溯，删除整条接缝

  3. 重复此过程，直至图像到达指定宽度
